🚀 Personal Portfolio | Interactive Experience
Este repositorio contiene el código fuente de mi portfolio personal, diseñado para demostrar mis habilidades en el desarrollo Full Stack con un fuerte énfasis en el Frontend creativo.

🛠️ Tecnologías y Herramientas
HTML5 & CSS3 Avanzado: Estructura semántica y diseño responsivo utilizando técnicas modernas como Glassmorphism y CSS Grid.

JavaScript (ES6+): Motor principal para la lógica de navegación, manejo de eventos y control de estado de la interfaz.

GSAP (GreenSock Animation Platform): Implementación de animaciones cinemáticas, efectos de parallax y revelado de secciones con alto rendimiento.

Lenis Scroll: Integración de scroll suave (smooth scroll) para mejorar la inmersión y la fluidez de la navegación.

Git & GitHub: Control de versiones y despliegue continuo.

✨ Características Principales
Universo Dinámico: Sistema de partículas y múltiples capas de estrellas optimizadas para evitar el lag de renderizado.

Navegación Inteligente: Header adaptativo que detecta la intención del usuario y scroll centrado automático para secciones críticas.

Optimización de Rendimiento: Uso de will-change, manipulación eficiente del DOM y gestión de opacidades para garantizar 60fps.
